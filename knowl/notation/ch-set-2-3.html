Note:
  1. Sets are denoted by capital letters, such as \(A, B, C, \ldots \) or \(X, Y, Z, \ldots\text{.}\)
  2. Elements of sets are generally denoted by small letters. e.g.: \(a, b, c, \ldots, etc.\)
  3. Elements are not repeated in a set e.g.: \(A = \{a,a,b,c\} = \{a,b,c\}\text{.}\)
  4. if \(a\) is an element of the set \(A\text{,}\) we write \(a \in A\text{.}\)
