Standard notations of some special sets:
\begin{align*} {\mathbb N} &= \{n: n \text{ is a natural number}\} = \{1, 2, 3, 4, 5, \ldots \}\\ {\mathbb W} &= \{w: w \text{ is a whole number}\} = \{0, 1, 2, 3, 4, 5, \ldots \}\\ {\mathbb Z} &= \{n : n \text{ is an integer} \} = \{\ldots, -2, -1, 0, 1, 2, \ldots \}\\ {\mathbb Q} &= \{r : r \text{ is a rational number}\} = \{p/q : p, q \in {\mathbb Z} \text{ where } q \neq 0\}\\ {\mathbb R} &= \{ x : x \text{ is a real number} \} = \{x :x \text {is rational or irrational number.} \}\\ {\mathbb E} &= \{ x : x \text{ is an even number} \} = \{\ldots, -6, -4, -2, 0, 2, 4, 6, \ldots \} \\ {\mathbb O} &= \{ x : x \text{ is an odd number} \} = \{\ldots, -3, -1, 1, 3, 5, \ldots \} \\ {\mathbb C} &= \{z : z \text{ is a complex number}\}\text{.} \end{align*}
