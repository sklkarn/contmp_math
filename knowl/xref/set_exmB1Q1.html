Example 2.9.18.

Find the union of each of the following pairs of sets:
  1. \(X = \{1, 3, 5\}\text{,}\) \(Y=\{1, 2, 3\}\)
  2. \(A = \{a, e, i, o, u\}\text{,}\) \(B=\{a, b, c\}\)
  3. \(A = \{x : x \text{ is a natural number and multiple of } 3 \}\text{,}\) \(B=\{x : x \text{ is a natural number less than } 6 \}\)
  4. \(A = \{x : x \text{ is a natural number and } 1 \lt x \leq 6 \}\text{,}\) \(B=\{x : x \text{ is a natural number and } 6 \lt x \lt 10 \}\)
  5. \(A = \{1, 2, 3\}\text{,}\) \(B=\phi\text{.}\)
