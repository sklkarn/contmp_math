    Section 4.6 Example B

    Example 4.6.41.

    Pascals Rule. Prove that
    \begin{equation*} ^nC_r + ^nC_{(r-1)} = ^{(n+1)}C_r \end{equation*}
    Solution.
    \begin{align*} ^nC_r + ^nC_{(r-1)} \amp = \frac{n!}{r!(n-r)!} + \frac{n!}{(r-1)!(n-r+1)!}\\ \amp = \frac{n!}{r(r-1)!(n-r)!} + \frac{n!}{(r-1)!(n-r+1)(n-r)!} \\ \amp = \frac{n!}{(r-1)!(n-r)!} \left( \frac{1}{r}+ \frac{1}{n-r+1}\right) \\ \amp = \frac{n!}{(r-1)!(n-r)!} \left( \frac{(n+1)}{r(n-r+1)}\right) \\ \amp = \frac{(n+1)n!}{r(r-1)!(n-r+1)(n-r)!} \\ \amp = \frac{(n+1)!}{r!(n+1-r)!} \\ \amp = ^{n+1}C_{r} \end{align*}

    Example 4.6.44.

    If
    \begin{equation*} ^nC_{(r-1)}:^nC_r:^nC_{(r+1)}=3:4:5 \end{equation*}
    find the value of \(n\) and \(r.\)
    Solution.
    \begin{align*} ^nC_r : ^nC_{(r+1)}\amp = 4:5 \\ \frac{^nC_r}{^nC_{(r+1)}} \amp = \frac{4}{5} \end{align*}
    From Theorem4.5.40
    \begin{equation*} \frac{^nC_{r}}{^nC_{(r+1)}} = \frac{r+1}{n-r} = \frac{4}{5} \end{equation*}
    or,
    \begin{equation*} 5(r+1) = 4(n-r) \end{equation*}
    \begin{equation} 4n-9r =5\tag{4.6.1} \end{equation}
    \begin{equation*} ^nC_{(r-1)}:^nC_{r} = \frac{3}{4} \end{equation*}
    \begin{equation*} \frac{^nC_{(r-1)}}{^nC_{r}} = \frac{3}{4} \end{equation*}
    \begin{equation*} \frac{^nC_{r}}{^nC_{(r-1)}} = \frac{n-r+1}{r} = \frac{4}{3} \end{equation*}
    \begin{equation} 3n-7r =-3\tag{4.6.2} \end{equation}
    On solving (4.6.1) and (4.6.2), we get -
    \(r=27\) and \(n=62.\)

    Example 4.6.45.

    If \(^{2n}C_3:^nC_3 = 11:1,\) find \(n.\)
    Solution.
    \begin{equation*} \frac{^{2n}C_3}{^nC_3} = \frac{11}{1} \end{equation*}
    or,
    \begin{equation*} \frac{ \frac{2n(2n-1)(2n-2)}{3!}}{ \frac{n(n-1)(n-2)}{3!}} = \frac{11}{1} \end{equation*}
    or,
    \begin{equation*} \frac{2n(2n-1)(2n-2)}{n(n-1)(n-2)} = \frac{11}{1} \end{equation*}
    or,
    \begin{equation*} \frac{2\times 2\cancel{n}(2n-1)\cancel{(n-1)}}{\cancel{n}\cancel{(n-1)}(n-2)} = \frac{11}{1} \end{equation*}
    or,
    \begin{equation*} \frac{4(2n-1)}{n-2} = 11 \end{equation*}
    or,
    \begin{equation*} n = 6. \end{equation*}

    Example 4.6.46.

    If \(^nP_r = ^nP_{r+1}\) and \(^nC_r = ^nC_{r-1}\text{,}\) Find \(n\) and \(r.\)
    Solution.
    \begin{align*} ^nP_r \amp = ^nP_{r+1} \\ \frac{\cancel{n!}}{(n-r)!} \amp = \frac{\cancel{n!}}{(n-r-1)!} \\ (n-r-1)! \amp = (n-r)! \\ \cancel{(n-r-1)}! \amp = (n-r)\cancel{(n-r-1)}! \end{align*}
    \begin{equation} n = 1+r \tag{4.6.3} \end{equation}
    also,
    \begin{align*} ^nC_r \amp = ^nC_{r-1} \\ \frac{\cancel{n!}}{r!(n-r)!} \amp = \frac{\cancel{n!}}{(r-1)!(n-r+1)!} \\ (r-1)!(n-r+1)! \amp = r!(n-r)! \\ \cancel{(r-1)}!(n-r+1)\cancel{(n-r)!} \amp = r\cancel{(r-1)!}\cancel{(n-r)!} \end{align*}
    \begin{equation} n = 2r-1 \tag{4.6.4} \end{equation}
    On solving (4.6.3) and (4.6.4), we get -
    \begin{equation*} r =2 \end{equation*}

    Example 4.6.47.

    Prove that
    \begin{equation*} ^{2n}C_n = \frac{2^n\times [1.3.5\ldots (2n-1)]}{n!} \end{equation*}
    Solution.
    From LHS,
    \begin{equation*} ^{2n}C_n = \frac{(2n)!}{n!(2n-n)!} = \frac{(2n)(2n-1)(2n-2)!}{n!n!} \end{equation*}
    \begin{equation*} = \frac{(2n)(2n-1)(2n-2)(2n-3) \ldots 4.3.2.1}{n!n!} \end{equation*}
    \begin{equation*} = \frac{(2n)(2n-2)(2n-4) \ldots 4.2 (2n-1)(2n-3) \ldots 5.3.1}{n!n!} \end{equation*}
    \begin{equation*} = \frac{2^n\times n(n-1)(n-2) \ldots 3.2.1 (2n-1)(2n-3) \ldots 5.3.1}{n!n!} \end{equation*}
    \begin{equation*} = \frac{2^n\times \cancel{n!}\times[1.3.5.\ldots (2n-1)] }{n!\cancel{n!}} \end{equation*}

    Example 4.6.48.

    If \(^{(n+2)}C_8 : ^{(n-2)}P_4 = \frac{57}{16}\text{,}\) find \(n.\)
    Solution.
    \begin{align*} \frac{^{(n+2)C_8}}{^{(n-2)}P_4} \amp =\frac{57}{16} \\ \frac{\frac{(n+2)!}{8!(n+2-8)!}}{\frac{(n-2)!}{(n-2-4)!}} \amp =\frac{57}{16} \\ \frac{(n+2)!}{8!\cancel{(n-6)!}} \times \frac{\cancel{(n-6)!}}{(n-2)!} \amp =\frac{57}{16} \\ \frac{(n+2)(n+1)n(n-1)\cancel{(n-2)!}}{8!\cancel{(n-2)!}} \amp =\frac{57}{16} \\ (n+2)(n+1)n(n-1) \amp = 21 \times 20\times 19 \times 18 \end{align*}
    \begin{equation*} \therefore \quad n=19. \end{equation*}

    Example 4.6.54.

    How many diagonals are there in a polygon of \(n\) sides?
    Solution.
    Polygon of \(n\) sides has \(n\) vertex. In polygon (number of sides = number of vertex).
    That is, total points on polygon = \(n.\)
    total line segments = \(^nC_2.\)
    number of diagonals = (total line segments)-(number of sides of polygon)
    \begin{equation*} or, \quad \text{ number of diagonals } = ^nC_2-n \end{equation*}
    \begin{equation*} = \frac{n!}{2!(n-2)!}-n = = \frac{n(n-1)\cancel{(n-2)!}}{2!\cancel{(n-2)!}}-n = \frac{n(n-1)}{2}-n \end{equation*}
    \begin{equation*} \therefore \quad \text{ number of diagonals } = ^nC_2-n = \frac{n}{2}[n-3]. \end{equation*}
